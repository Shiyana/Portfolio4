# StandardScaler and KMeans in action
from sklearn.preprocessing import StandardScaler
from sklearn.cluster import KMeans
import pandas as pd

# Load data
df = pd.read_csv('data.csv')
scaler = StandardScaler()
scaled_data = scaler.fit_transform(df)

# Apply KMeans
kmeans = KMeans(n_clusters=3, random_state=42)
kmeans.fit(scaled_data)
df['Cluster'] = kmeans.labels_

Challenges & Solutions

Each challenge faced in the data pipeline was methodically addressed:

  • Missing Data: Used mean/median imputation and KNNImputer depending on feature correlation.
  • Noisy Data: Smoothing via rolling averages and noise reduction using filtering methods like Savitzky-Golay.
  • Class Imbalance: Utilized SMOTE to generate synthetic samples and balanced the dataset.
  • Interpretability: Used cluster center coordinates and visualization tools to interpret each group meaningfully.
