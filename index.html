Overview

This artifact demonstrates the application of K-Means clustering to uncover hidden patterns in customer segmentation data. The project emphasizes the power of unsupervised learning techniques in extracting insights from unlabelled data and improving business decision-making processes.

Process

Step 1: Data Cleaning – removed null values and normalized numerical columns using MinMaxScaler.
Step 2: Feature Engineering – selected relevant features including income, spending score, and age.
Step 3: Determined optimal number of clusters using Elbow and Silhouette methods.
Step 4: Applied K-Means with k=3 and assigned labels.
Step 5: Visualized clusters using 2D scatter plots and silhouette graphs.

Challenges & Solutions

  • Determining k: Addressed using both Elbow and Silhouette methods to cross-validate cluster quality.
  • Outlier Handling: Detected and removed outliers to ensure clean separation between clusters.
  • Feature Selection: Iteratively refined features based on domain relevance and correlation checks.
