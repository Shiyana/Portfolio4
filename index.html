Overview

This artifact explores unsupervised learning via the K-Means clustering algorithm. It includes insights from a series of chatbot-guided data science scenarios tackling missing data, noisy signals, and class imbalance. Each of these challenges was encountered and resolved using Python-based preprocessing and model evaluation tools.

The overview also highlights the ethical and leadership aspects of deploying AI solutions in education and healthcare. By understanding bias in data and decision-making, the artifact reflects a holistic view of how AI can serve equity and inclusion when used responsibly.

Process

From loading the dataset to final evaluation, the K-Means process involved multiple technical steps:

# Data Loading and Scaling
from sklearn.preprocessing import StandardScaler
from sklearn.cluster import KMeans
import pandas as pd

df = pd.read_csv('data.csv')
scaler = StandardScaler()
scaled_data = scaler.fit_transform(df)

# K-Means Application
kmeans = KMeans(n_clusters=3, random_state=42)
kmeans.fit(scaled_data)
df['Cluster'] = kmeans.labels_

Challenges & Solutions

  • Scenario 1 - Missing Data: Used mean and KNN imputation to preserve distributional integrity.
  • Scenario 2 - Noisy Data: Applied rolling mean and Savitzky-Golay filters to smooth out spikes in sensor data.
  • Scenario 3 - Imbalanced Data: Implemented SMOTE to synthetically balance minority class samples.
  • Model Bias: Conducted fairness audits to detect and correct outcome skew across subgroups.
