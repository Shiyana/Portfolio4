Artifact 4: Clustering Analysis with K-Means

Overview

This artifact showcases a comprehensive clustering analysis project using the K-Means algorithm, integrated with ethical reflections and chatbot-guided scenario exploration. The activity began with an AI chatbot guiding me through three core machine learning challenges: missing data, noisy data, and imbalanced datasets. Each scenario offered real-world complexity and required in-depth application of preprocessing techniques and model tuning to improve results.

Following this, I developed a clustering pipeline using a customer segmentation dataset. The project spanned from data cleaning and exploration to clustering using the Elbow method and Silhouette scores, resulting in valuable insights into behavioral patterns. Moreover, I incorporated my leadership learning reflection on navigating human bias, connecting my technical growth with responsible AI practices in the field of education and healthcare. This artifact exemplifies how technical, ethical, and analytical dimensions combine to build meaningful machine learning solutions.

Process

Step 1: Data Loading & Exploration - Loaded the dataset, assessed data types, and visualized distributions. Identified missing values and outliers.
Step 2: Data Cleaning - Applied median imputation and z-score-based outlier removal. Used standard scaling for normalization.
Step 3: Feature Engineering - Explored relationships between features using pairplots, correlation heatmaps, and domain context.
Step 4: Clustering Algorithm - Applied K-Means using scikit-learn. Used Elbow Method and Silhouette Score to choose optimal k (k=3).
Step 5: Visualization - Visualized results with 2D scatterplots. Created cluster profile summaries for interpretation.

Challenges & Solutions

  • Challenge: Determining the correct number of clusters.
    Solution: Combined Elbow Method with Silhouette Scores to validate k=3.
  • Challenge: Handling overlapping data clusters.
    Solution: Used PCA for dimensionality reduction to separate features more clearly.
  • Challenge: Interpreting cluster meanings.
    Solution: Analyzed centroid statistics and matched to business logic.
