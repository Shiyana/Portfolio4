Overview

Project Summary: This artifact demonstrates a comprehensive clustering analysis using K-Means. It combines technical execution, leadership reflection, and ethical considerations in AI. Throughout the project, I tackled three major scenarios in ML: missing data, noisy data, and class imbalance.

  • Scenario 1: Visualized missing data using heatmaps, applied multiple imputations, and evaluated their effect on clustering coherence.
  • Scenario 2: Compared median filtering and exponential smoothing; validated using residual plots and signal-to-noise ratios.
  • Scenario 3: Used `imbalanced-learn`’s SMOTE and ADASYN for oversampling; tracked F1 improvements in minority prediction.

Process

  • Step 1: Loaded data from CSV using `pandas.read_csv()`.
  • Step 2: Cleaned with outlier removal (IQR filtering) and normalization using `StandardScaler`.
  • Step 3: Chose top features based on variance thresholding and correlation matrices.
  • Step 4: Performed PCA for visualization and used Elbow method to determine k=3.
  • Step 5: Trained KMeans, plotted clusters using matplotlib with decision boundaries.
