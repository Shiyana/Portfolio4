Process

This section provides a step-by-step walkthrough of how I executed clustering on a cleaned dataset:

  • Data Cleaning: Addressed missing values using KNN imputation and dropped high-NA columns.
  • Feature Engineering: Normalized continuous data, encoded categorical columns, and selected top features via correlation.
  • Dimensionality Reduction: Applied PCA for 2D visualization and reduced noise using low-variance filtering.
  • Clustering: Used KMeans (k=3), confirmed optimal k via the Elbow and Silhouette methods.
  • Visualization: Visualized clusters and centroids in 2D PCA space.

Challenges & Solutions

  • Missing Data: Resolved using a hybrid of mean/mode imputation and KNNImputer based on feature type and correlation with target.
  • Noisy Data: Detected using Z-score method; applied exponential smoothing to sensor-derived features.
  • Bias: Cluster interpretation revealed possible demographic bias. Resampled subgroups for fair representation.
  • Scalability: Addressed memory constraints by using mini-batch KMeans on larger versions of the dataset.
