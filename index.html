Overview

This artifact showcases my hands-on application of clustering techniques using the K-Means algorithm. Alongside, I engaged in a multi-scenario chatbot activity involving real-world data issues like missing values, noisy data, and class imbalance.

The activity helped solidify my ability to analyze, clean, and model data under different conditions. It required critical thinking, the application of proper imputation and noise reduction methods, and strategies like SMOTE and evaluation metrics (F1-score, AUC) for imbalanced datasets.

This combined learning experience demonstrates both technical proficiency and reflective practice across the Machine Learning Fundamentals course.

Process

Step 1: Cleaned missing values and scaled numerical features.
Step 2: Explored features like income and spending behavior.
Step 3: Used Elbow and Silhouette methods to determine optimal clusters.
Step 4: Performed K-Means clustering with k=3.
Step 5: Created visualizations to evaluate segmentation quality.

Challenges & Solutions

  • Unclear clustering patterns: Mitigated with multiple evaluation metrics.
  • Outliers: Filtered prior to training using IQR and Z-score methods.
  • Cluster interpretation: Used feature distributions to label clusters meaningfully.

Key Takeaways

  • Learned to apply clustering to real customer segmentation problems.
  • Developed skills to validate unsupervised learning outcomes.
  • Explored SMOTE and outlier techniques in the chatbot assignment.
  • Improved confidence working with raw, incomplete, and noisy data.

Reflection

This project helped me develop an integrated understanding of unsupervised learning and real-world data cleaning. It made me aware of how preprocessing affects downstream analysis, model robustness, and fairness.

The chatbot activity taught me to think critically about model assumptions, how to prevent bias, and how to approach data limitations systematically. I now feel ready to explore clustering in industry settings and combine it with customer behavior insights.

Conclusion & Future Work

I plan to compare K-Means with hierarchical and DBSCAN clustering. I'd also like to experiment with dimensionality reduction using PCA and t-SNE before clustering to gain deeper insights from complex datasets.
